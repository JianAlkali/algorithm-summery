[[数论]]

最坏$O\left(\sqrt{n}\right)$，平均$O\left(\log n\right)$地计算一个数的所有因数。相比枚举因子，它的好处是平均复杂度成为了$O\left(\log n\right)$，坏处是预处理质数需要$O\left(N\right)$，所以N会很大时不能用。前置代码：带质因数分解的质数筛（[[欧式筛]]）。N较大时请用枚举法。1e9范围内，一个数的因数数量最大为1344（$<\frac{\sqrt{10^{9}}}{20}$）。
```cpp
int dis[N], dsz=0, dn[32], dc[32], dnsz=0;
// 得到的所有因数位于大小为dsz的dis里。大小为dnsz的dn和dc存的是其分解的质因数和该质因数的次数
auto in_dis=[&](auto&& in_dis, int i, int now)->void{
        if(i==dnsz){
            dis[dsz++]=now; return;
        }
        int ddn=dn[i], ddc=dc[i];
        for(int j=0;j<=ddc;j++){
            in_dis(in_dis,i+1,now);
            now*=ddn;
        }
    };
    auto cal_dis=[&](int a){ //调用此函数获取所有因数
        if(a==1){
            dis[0]=dsz=dn[0]=dc[0]=dnsz=1; return;
        }
        dnsz=0; dsz=0;
        int pdis=-1;
        while(a!=1){
            int dis=ppm[a];
            a/=dis;
            if(dis!=pdis){
                pdis=dis;
                dc[dnsz]=1;
                dn[dnsz++]=dis;
            }else
                dc[dnsz-1]++;
        }
        in_dis(in_dis,0,1);
    };
```
