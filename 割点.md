[[_图论]]

割点是相对于无向图而言的，是删除该点后会导致一个连通图不连通的点。
相关知识：割点/割边/双联通分量/缩点计算的复杂度都为$O\left(n+m\right)$。割点也叫必经点，割边(桥)也叫必经边。点双联通分量即不存在割点的连通图，边双联通分量即不存在割边的连通图。它们的核心都是tarjan。一个点可能属于多个点双连通分量，但只会属于一个边双连通分量。
割点：`u`是有两个或以上子树的树根 or `u`非树根且存在树边$u→v: low\left[v\right]≥dfn\left[u\right]$。
割边：树边$u→v: low\left[v\right]>dfn\left[u\right]$，在处理重边时需要特殊判断。
边双缩点：无向图变树，点双缩点：将点双看做方点，割点看做圆点相互连接成树。
```cpp
struct E{
    int v,nxt;
}e[N<<1];
int head[N], ec=0;
void add(int u,int v){
    e[++ec]={v,head[u]};
    head[u]=ec;
}
int dfn[N], low[N], dfnt=0;
int ans[N];
void solve(){
    int n,m;
    cin>>n>>m;
    ec=dfnt=0;
    for(int i=0;i<=n;i++){
        dfn[i]=low[i]=head[i]=ans[i]=0;
    }
    for(int i=0,u,v;i<m;i++){
        cin>>u>>v;
        add(u,v);
        add(v,u);
    }
```

    auto tar=[&](auto&& tar,int u,int f)->void {
        dfn[u]=low[u]=++dfnt;
        int son=0;
        for(int i=head[u];i;i=e[i].nxt){
            int v=e[i].v;
            if(v==f) continue;

                low[u]=min(low[u],low[v]);
                if(low[v]>=dfn[u] && f) ans[u]=1;
                son++;
            }else
                low[u]=min(low[u],dfn[v]);
        }
        if(!f && son>=2) ans[u]=1;
    };

        if(!dfn[i]) tar(tar,i,0);

        ac+=ans[i];

        if(ans[i]) cout<<i<<' ';
}
