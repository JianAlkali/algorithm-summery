[[_数论]]

$O\left(N\right)$计算所有$≤N$的数中有哪些质数，除此之外还有许多变种，可以同时记录额外的信息。
## 标准版
大小为`pmn`的`pm`存质数，同时记录该位置的最小质因数（由`ppm`表示）
```cpp
int pm[N], pmn=0, ppm[N];
inl void init(){
    ppm[1]=ppm[0]=1;
    for(int i=2;i<N;i++){
        if(!ppm[i]){
            ppm[i]=i; //注释掉这一行，则变为有且仅有质数的ppm为0
            pm[pmn++]=i;
        }
        for(int j=0;j<pmn && i*pm[j]<N;j++){
            ppm[i*pm[j]]=pm[j];
			//使用 ppm[i*pm[j]]=ppm[i]; ppm记录的则是最大质因数
            if(i%pm[j]==0)
                break;
        }
    }
}
```

## 异或哈希
预处理后，$O\left(m\right)$计算多个数相乘，其质因子是否为均为偶数次幂：
```cpp
int pm[N], pmn=0, ppm[N], ha[N];
mt19937_64 rng(233);
inl void init(){
    for(int i=2;i<N;i++){
        if(!ppm[i]){
            ha[i]=rng();
			//…
        }
        for(int j=0;j<pmn && i*pm[j]<N;j++){
            ha[i*pm[j]]=ha[i]^ha[pm[j]];
			//…
        }
    }
}
```
