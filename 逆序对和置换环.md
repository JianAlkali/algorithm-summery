逆序对指满足$1≤l<r≤n，a_{l}>a_{r}$的`l`、`r`。置换环是对一个排列，将值看成出边，形成的环，如$a_{1}=2,a_{2}=3,a_{3}=1$或$a_{1}=1$。这两个对于一个排列（或者离散化后为排列）来说，有一些特别的关系。下称逆序对数为`v`，置换环数为`c`，奇偶性为`p`，我们也称排列的奇偶性为$v_{p}$。
交换相邻两个逆序元素：$v-=1,  c±=1\left(同环为-，异环为+\right)$。
交换相邻元素：$v±=1,  c±=1\left(同上\right)$。
交换任意两个元素：$v±=1-2t,  v_{p}⊕=1,  c±=1$
区间$\left[l,r\right]$循环左移`k`次：$v_p \oplus = (len-1)_p \ \&\ k_p, \quad c_p \oplus = (len-1)_p \ \&\ k_p$
区间$\left[l,r\right]$翻转：$v=\frac{k⋅\left(k-1\right)}{2}-v_{ori}$, $v_{p}⊕=\left(\frac{k⋅\left(k-1\right)}{2}\right)_{p} ≔kmod4≡2or3$, $c_{p}同v_{p}$
## 归并排序计算逆序对数量$O\left(n\log n\right)$：
```cpp
int a[N], tmp[N];
auto merge=[&](auto&& merge,int l,int r)->int {
        if(l==r) return 0;
        int m=(l+r)>>1, cntv=0;
        cntv+=merge(merge,l,m);
        cntv+=merge(merge,m+1,r);
        int i=l,j=m+1,k=l;
        while(i<=m&&j<=r){
            if(a[i]<=a[j]) tmp[k++]=a[i++]; //使用<则将相同大小也统计（在计算不递增/不递减的时候有用）
            else{
                cntv+=j-k;
                tmp[k++]=a[j++];
            }
        }
        while(i<=m) tmp[k++]=a[i++];
        while(j<=r){
            cntv+=j-k;
            tmp[k++]=a[j++];
        }
        copy(tmp+l,tmp+r+1,a+l);
        return cntv;
    };
```
## 统计置换环$O\left(n\right)$：
```cpp
    fill(vis,vis+n+1,false);
    int cntc=0;
    for(int i=1;i<=n;i++){
        if(vis[i]) continue;
        int j=i,k=i;
        do{
            vis[j]=true;
            j=c[j];
            k=c[c[k]];
        }while(j!=k);
        cntc++;
    }
```
