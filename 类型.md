`mt19937_64` 一个类型，返回64位随机数 使用例：`mt19937_64 rng(233); int r=rng();` 32位的版本为`mt_19937`
`multimap<K,V>` 一个类型，与`map`不同的地方：不支持`[]`。插入相同键，后插的排在后面。 `.find(k)→it`返回的是第一个`key=k`的键值对。`.count(k)→size_t` 返回的是`key=k`的`key`数量。`.equal_range(k)→pair<begin,end>` 返回`key=k`的开始和结束迭代器，遍历其：`for(auto it=res.first; it!=res.second; ++it){} `。`.erase(k)`删除所有`key=k`的；若要删除单一个元素，使用` .erase(.find(k))`。
`priority_queue<Ty, vector<Ty>, cmp>`的比较函数写法：
```cpp
struct Pos{
    int x,y,fc; // 我们自己的数据
};
struct Cmp{ // 我们想按pair.first升序排列（小顶堆）
    bool operator()(const pair<int,Pos>& l, const pair<int,Pos>& r){
        return l.first>r.first;
    }
};
priority_queue<pair<int,Pos>,vector<pair<int,Pos>>,Cmp> pq;
```
