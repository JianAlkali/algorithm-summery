[[动态规划]]

使用STL[[位运算]]函数可实现一些操作。
拿取mask的每一位：
```cpp
    for(int t=mask;t;t&=t-1){
        int b=__builtin_ctzll(t);
        // 使用..
    }
```

枚举mask的所有非空子集（包含空集只需要改break位置）：
```cpp
for(int sub=mask;sub;sub=(sub-1)&mask)
```

Gosper’s Hack（Gosper枚举法）：
即枚举固定的k=popcount值下的m位所有二进制位组合，$O\left(1\right)$每次，共$O\left(C\left(m,k\right)\right)$
三种写法完全等效，推荐用函数写法：
```cpp
inline int gnxt(int s){
    int c = s & -s;
    int r = s + c;
    return (((r ^ s) >> 2) / c) | r;
}
```

```cpp
#define low(x) ((x)&(-(x)))
#define gnxt(x) \
    ( ((x)+low(x)) | \
    ((( ((x)+low(x)) ^(x)) >>2 ) /low(x) ) )
```

```cpp
    int x = s & -s; // 直接写在for循环尾部
    int y = s + x;
    s = ((s & ~y) / x >> 1) | y;
```

调用（其中k不能为0，注意1写成1ll）：
```cpp
for(int s=(1LL<<k)-1; s<(1LL<<m); s=gnxt(s))
```

mask中选k个：
也是利用Gosper，复杂度为理论最优$O\left(k⋅ C\left(m,k\right)\right)$。
```cpp
    vi bits;
    for(int i=mask;i;i&=i-1)
        bits.emplace_back(__builtin_ctzll(i));
    int m=bits.size();
    for(int s=(1ll<<k)-1;s<(1ll<<m);s=gnxt(s)){
        int real=0, t=s;
        while(t){
            real |= 1ll<<bits[__builtin_ctzll(t)];
            t&=t-1;
        }
        // 使用real
        cout<<real<<'\n';
    }
```
