[[动态规划]]

（1）check在[low, high)上返回值分别为F…FFTT…T，二分结束后l将指向第一个T，全为F时l指向high。
```cpp
    int l=low, r=high;
    while(l<r){
        int m=(l+r)>>1;
        if(check(m)) r=m;
        else l=m+1;
    } // l->first true
```

（2）check在[low, high)上返回值为T…TTFF…F，二分结束后l指向第一个T或high。
```cpp
    int l=low, r=high;
    while(l<r){
        int m=(l+r)>>1;
        if(check(m)) l=m+1;
        else r=m;
    } // l->first false
```

对于（1）（2），如果要找最后一个F/T，l—即可，此时l最终的取值为[low-1, high)，注意范围。
（3）lower_dound(begin, end, tar, func)，找到[begin,end)范围内第一个不小于tar的数。func的写法：bool comp(int val, int tar)，默认使用std∷less（return val<tar）。
*一般来说，stl传函数的写法有严格弱序的bool comp(int a, int b){return a<b;}和对于特定算法的bool pred(int a){return 条件a?;}。多数情况下写bool(int, int)总没错，且该函数不能用≥、≤。
