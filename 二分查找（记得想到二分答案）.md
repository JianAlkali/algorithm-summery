[[_动态规划]]

## 1.前F后T时
`check`在`[low, high)`上返回值分别为`F…FFTT…T`，二分结束后`l`将指向第一个`T`，全为`F`时`l`指向`high`。
```cpp
    int l=low, r=high;
    while(l<r){
        int m=(l+r)>>1;
        if(check(m)) r=m;
        else l=m+1;
    } // l->first true
```
## 2.前T后F时
`check`在`[low, high)`上返回值为`T…TTFF…F`，二分结束后`l`指向第一个`T`或`high`。
```cpp
    int l=low, r=high;
    while(l<r){
        int m=(l+r)>>1;
        if(check(m)) l=m+1;
        else r=m;
    } // l->first false
```

对于`1.` `2.`，如果要找最后一个`F`or`T`，退出while后`l--`即可，此时l最终的取值为`[low-1, high)`，注意范围。
## 3.直接调库
`lower_dound(begin, end, tar, func)`，找到`[begin,end)`范围内第一个不小于`tar`的数。其`func`参数的写法：`bool comp(int val, int tar)`，默认使用`std∷less`（`return val<tar`）。

**一般来说，stl传函数的写法有严格弱序的`bool comp(int a, int b){return a<b;}`和对于特定算法的`bool pred(int a){return 条件a?;}`。多数情况下要传函数时写`bool(int, int)`总没错，且该函数不能用≥、≤（违背严格弱序）。**
