[[_数论]]

任何一个正整数`x`，都可用4个平方数表示$x=a^{2}+b^{2}+c^{2}+d^{2}$，而且能用3个平方数表示，当且仅当$x≠4^{a}⋅\left(8b+7\right)$时。可 $O\left(\sqrt{n}⋅\log N\right)$计算出表示x所需最少的平方数的数量：
```cpp
int sqr[N], sqrsz=0;
inl void init(){
    for(int i=1;i*i<N;i++)
        sqr[sqrsz++]=i*i;
}
inl void solve(){
    int x;
    cin>>x;
    if(binary_search(sqr,sqr+sqrsz,x)){
        cout<<1<<'\n'; return;
    }
    int y=x;
    while((y&3)==0) y>>=2; // 除去4^a logn
    if(y%8==7){
        cout<<4<<'\n'; return;
    }
    for(int i=1;i*i<x;i++){ //sqrt(n)log(N)
        if(binary_search(sqr,sqr+sqrsz,x-i*i)){
            cout<<2<<'\n'; return;
        }
    }
    cout<<3<<'\n'; return;
}
```
